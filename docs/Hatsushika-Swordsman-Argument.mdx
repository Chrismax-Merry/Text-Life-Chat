演武仪典在即，三月七受邀成为彦卿、云璃的剑术学徒。而你将作为她的助教，规划练剑日程，帮助她取得剑斗胜利。

在 [初花习剑录 ] 中，你将作为剑术助教，安排三月七的练剑日程，并帮助她领悟剑招，在十五日后击败斯科特。

### 情景对话
青雀：你是不是觉得凌晨的手机很好玩？你是不是觉得早上的觉真的很好睡？你是不是成天赚着窝囊费，还要听着领导说着你干啥都多余……

青雀：你试着想想你上班之外的生活，是不是脑海里只能浮现两个字？

三月七：加 ~ 班 ~ ！

### 与青雀交谈
> 经过十五日的剑术练习与奇遇，三月七与你所想一致，仅仅是剑斗胜利已经不能满足了。十五日中，你们多次与青雀一同经历「江湖奇遇」，也许可以向青雀寻求一点建议。

```html
三月七：马上就要和斯科特对决了，青雀，你知道我为什么来找你吗？

青雀：这里有你，有我，有开拓者，这不显而易见了吗？

选择项：一起商量击败斯科特的办法。

青雀：我可不会什么剑斗啊，如果不是打牌的话，我好像没有什么能够帮你的。

三月七：等等！最近我不是经常和你一起玩吗，其实我有了不少心得体会。比如，最近我一直在思考，仅仅用剑击败斯科特是不够的——

选择项：最近？一直？在思考？

三月七：…对啊。你不会真的以为我傻了吧唧的吧！

三月七：你们看，虽然斯科特会道歉，但他是真心实意地感到愧疚吗？

青雀：那只是愿赌服输吧——道歉也好，学猪叫也好，估计他都不是心甘情愿的。

选择项：他之前的狗叫也不是真的道歉。

三月七：没错，他就是想扬眉吐气才把比斗的地点选在金人巷。

青雀：…有点道理。虽然我没有那么想凑热闹啦，我就帮你参考一下吧。

三月七：开拓者，你也可以讲讲之前金人巷的事。

旁白：青雀饶有兴致地听起斯科特事件相关的始末。等到你们讲完时，你注意到青雀的脸上好像有些怒气。

青雀：如果喜欢工作是一种病，那这位斯科特简直应该待在重症监护室里躺着！只是喜欢就算了，还要出来危害社会，这可不行！

选择项：那你有什么建议吗？

青雀：我没有什么建议，只有评价，我的评价是「上班上的」。

青雀：走吧，三月，我们去会会他！

三月七：欸？！
```

### 前往金人巷，参加约定的决斗
> 青雀认定斯科特是「过度喜欢上班给他人造成麻烦的人」，她主动请缨，要会会这斯科特。
>
> 青雀应该不会剑术吧…难不成去讲道理去了？出发吧，先到金人巷再说。
```HTML
斯科特：瞧瞧，这不是我们的女侠三月七吗？你身边的，这是……

青雀：在你们正式开始比斗之前，我必须要跟你聊几句。

斯科特：哈——？！你又是谁啊，怎么，你们仙舟的人都这么喜欢不请自来吗？

青雀：你不认识我没有关系，开拓者，介绍一下。

选择项：这是我们帝玉集团的青总！

青雀：你别吓着他了，我平时主要担任太卜司的掌门工作。

斯科特：太卜司的卜者而已，有什么了不起吗，我还是星际和平公司的专员呢！你就说说吧，你是手下比我多，还是你挣的信用点比我多，你有什么了不起的。

青雀：你这人年纪不小，但是精神状态还是刚走进职场的小白一样——将几乎全部的时间用在了工作之上，美其名曰「热爱工作」。

斯科特：…你、你闭嘴！

青雀：我猜你喜欢将自己代入乐子人的身份，到处口出狂言，为的就是假装客观，以便在任何时候做出事不关己的姿态。

斯科特：够、够了——！你闭嘴，你又怎么会理解我！别说的自己好像什么都清楚一样！

三月七：精彩，第一次看到斯科特才说三两句话就被激怒了。

斯科特：差点就被你给绕进去了。今天本来是…是来决斗的，不是来和你辩经的。

选择项：斯科特被戳到痛处了。

三月七：斯科特没有办法反驳。

三月七：就是这个气势，现在正是趁热打铁的时候！
```

### 与斯科特交谈
> 说服斯科特
```html
青雀：我问你，公司是你家吗？

斯科特：公司当然是我家，公、公司里有我这么多的好兄弟…！你知道公司的架构吗？「人才激励部」的同事让公司比家还温暖！

选择项：要求斯科特扪心自问！

开拓者：你再好好想想，你把公司当作家，公司有把你当作家人吗？你爱公司，那公司爱你吗？

斯科特：公司，爱不爱我…公司又不是人，它不用爱我！

三月七：所以，公司就不是你家！别再骗自己了。

青雀：我听说庇尔波因特最低职级的员工，只要一想起公司的科技与财富，也会不由自主地挺起胸膛…但是你仔细想想，公司的一切成就和财富跟你有任何关系吗？

斯科特：当、当然有关系…了！

三月七：那你说说看，公司迄今为止哪一项成就与你有关？

斯科特：当初金人巷的业务我有参与…眼下这批公司运输船也是我在负责…虽然这些事情好像也不大，但是少了我这些事情就办不成！

选择项：质疑斯科特的不可取代。

开拓者：当初你离开后，金人巷业务推进得更好了。而你现在负责的运输船，真的没了你就不行吗？

斯科特：金、金人巷那是意外，要是听我的，公司早就拿下了！再说了，这运、运输船肯定离了我不行！

三月七：我听说公司每年入职的人和离开公司的人，比热夜之都的总人口还多…你不做，有的是斯科特能做。

青雀：你是不是觉得凌晨的手机很好玩？你是不是觉得早上的觉真的很好睡？你是不是成天赚着窝囊费，还要听着领导说着你干啥都多余……

斯科特：别、别说了，别说了！

青雀：你试着想想你上班之外的生活，是不是脑海里只能浮现两个字？

三月七：加~班~！

斯科特：别说了，求求你，别说了——

青雀：按照我的工作哲学，我其实也不想说「你不重要」，相反我会说「你很重要」。打工人，你要为自己好好生活而骄傲，而不是其他的任何事情。

斯科特：我、我根本没有自己的生活！

选择项：适当的沉默。

旁白：你沉默不语，流露出哀伤的眼神。如果说眼神有温度，那么你的爱怜可以孵化小鸡。

斯科特：求求你了，说两句吧，不要让我觉得自己很可怜。

三月七：没有自己的生活，你真的很可怜。

斯科特：啊啊啊啊——！

青雀：你自命不凡，蔑视他人，没有时间投入生活，缺乏共情能力，害怕没得到尊重，妄图靠着公司给的身份建立自信，你甘心吗——

斯科特：呜啊啊啊啊——也不用说的这么过分吧，我也是很努力地在生活了，我也是一个平凡人，你讲这些，难道自己的心就不会痛吗？

青雀：我当然会痛，因为我也是一个平凡人。现在，斯科特，欢迎你回到平凡人的世界。

青雀：不夜侯的小老板娘梦茗，她辛苦地备好茶水，安排好桌椅，就是为了让你我这样身不由己的打工人能够适当休息，找回属于自己的生活！

青雀：梦茗小姑娘不也是你吗？你辛辛苦苦做好自己的事，被没事找事的领导推翻桌子，打碎茶杯，还说你没有给他「五彩斑斓的黑」，你能受得了吗？

三月七：我们本想用剑斗的方式让你去给她道歉，但这是不够的。

斯科特：我需要发自内心地向她道歉。

斯科特：你们不要说了，我都明白，这场剑斗我们就不要继续了。我要去向不夜侯的小老板娘道歉，我要向所有认真生活的人道歉。

斯科特：谢谢你们，差点我就和你们交手了，我真是不应该啊。

斯科特：从明天起，做一个幸福的人，喝茶看报，漫游宇宙。

斯科特：不说了，我要去向小老板娘道歉了。谢谢你们，让我思考了不少…这破公司的工作，哎，我得花些时间好好想想。

斯科特：再见！

旁白：斯科特带着他的好兄弟们离开了金人巷，向不夜侯的店主梦茗道了歉。

青雀：哎，好累，我好久都没有这么累了。三月，你们得请我吃点好的。

三月七：当然当然，多谢青雀大师。

一段小插曲结束了，而三月七的剑术学习还将继续。
```
